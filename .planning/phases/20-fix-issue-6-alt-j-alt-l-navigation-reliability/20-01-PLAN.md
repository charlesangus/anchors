<objective>
Fix Alt+J, Alt+L, and Alt+Z navigation reliability by applying QTimer deferral (same pattern as AnchorNavigatePlugin.invoke) and fix get_links_for_anchor to find links inside Group nodes.

Purpose: These three navigation commands call DAG viewport operations synchronously within the menu callback. On Nuke 15.x (especially macOS), Qt focus has already shifted by the time the callback returns, so nuke.zoom/zoomToFitSelected targets the wrong panel or silently fails. The fix is to defer viewport operations with QTimer.singleShot(0, ...) to let Qt settle first — exactly as AnchorNavigatePlugin.invoke() already does. Additionally, get_links_for_anchor misses links inside Groups because it calls nuke.allNodes() without recurseGroups=True.

Output: Updated anchor.py with deferred navigation + recurse-group fix; updated tests verifying the new behavior.
</objective>

```python
    Saves the current DAG viewport first so Alt+Z (navigate_back) can return.
    Defers viewport operations via QTimer.singleShot(0, ...) so Qt focus

        return

    if anchor_node is None:
        return

        _save_dag_position()
        navigate_to_anchor(anchor_node)
```

```python
def cycle_next_link():
    """Navigate to the next link node referencing the selected anchor.

    On first invocation (or when the selected anchor changes), saves the DAG
    position, collects all links for the anchor sorted by name, and zooms to
    the first one.  Each subsequent invocation advances to the next link.
    After the last link, navigates back to the anchor and resets the cycle.

    Viewport operations are deferred via QTimer.singleShot(0, ...) so Qt focus
    settles back to the DAG panel before nuke.zoom runs.

    Silent no-op when:
    - The plugin is disabled
    - No nodes are selected
    - The selected node is not an anchor
    - The anchor has no link nodes
    """
    global _cycle_anchor_name, _cycle_links, _cycle_link_index

    if not prefs.plugin_enabled:
        return
    selected_nodes = nuke.selectedNodes()
    if not selected_nodes:
        return
    first_selected_node = selected_nodes[0]
    if not is_anchor(first_selected_node):
        return

    anchor_name = first_selected_node.name()
    links = sorted(get_links_for_anchor(first_selected_node), key=lambda n: n.name())
    if not links:
        return

    if anchor_name != _cycle_anchor_name or links != _cycle_links:
        _cycle_anchor_name = anchor_name
        _cycle_links = links
        _cycle_link_index = 0
    else:
        _cycle_link_index += 1

    if _cycle_link_index >= len(_cycle_links):
        _cycle_anchor_name = None
        _cycle_links = []
        _cycle_link_index = 0

        def _deferred_back():
            navigate_back()

        QtCore.QTimer.singleShot(0, _deferred_back)
        return

    target_link = _cycle_links[_cycle_link_index]
    # Capture current selection synchronously before deferring
    saved_selection = nuke.selectedNodes()

    def _deferred_zoom():
        if _cycle_link_index == 0:
            _save_dag_position()
        nukescripts.clear_selection_recursive()
        target_link["selected"].setValue(True)
        nuke.zoomToFitSelected()
        nukescripts.clear_selection_recursive()
        for node in saved_selection:
            node["selected"].setValue(True)

    QtCore.QTimer.singleShot(0, _deferred_zoom)
```

```python
def navigate_back():
    """Restore the DAG to the position saved before the last Alt+A jump.

    Silent no-op if no position has been saved yet. Consumes the slot --
    subsequent calls are no-ops until the next navigate-to-anchor jump.
    Defers the viewport zoom via QTimer.singleShot(0, ...) so Qt focus
    settles back to the DAG panel before nuke.zoom runs.
    """
    if not prefs.plugin_enabled:
        return
    global _back_position
    if _back_position is None:
        return
    zoom_level, center_xy = _back_position
    _back_position = None

    def _deferred_restore():
        nuke.zoom(zoom_level, center_xy)
        nukescripts.clear_selection_recursive()

    QtCore.QTimer.singleShot(0, _deferred_restore)
```
