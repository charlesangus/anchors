<objective>
Gate Dot anchor detection on label font size (minimum 33) so that small-labelled Dots are not treated as anchors or shown in navigation. Add a new "Label (Small)" command at font size 33 as the minimum anchor-qualifying label size.

Purpose: Users label Dots for organizational notes at small font sizes. These should not pollute the anchor navigation list or be treated as anchor dots. Only deliberately large-labelled Dots (font size >= 33) should qualify as anchors.

Output: Modified is_anchor() with font size gate, new DOT_LABEL_FONT_SIZE_SMALL constant, new create_small_label() function, new menu entry, and tests.
</objective>
