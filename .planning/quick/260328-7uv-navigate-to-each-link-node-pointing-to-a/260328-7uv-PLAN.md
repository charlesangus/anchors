```python
# Cycle-links state: tracks which anchor we are cycling and the current index
_cycle_anchor_name = None  # anchor node name during active cycle, or None

```

Implement `cycle_links_for_anchor()` in anchor.py:

1. Early return if `prefs.plugin_enabled` is False.
2. Get `nuke.selectedNodes()`. If empty, return.
3. Take the first selected node. If not `is_anchor(node)`, return.
4. Call `get_links_for_anchor(node)` to get the link list. If empty, return.
5. Sort the link list deterministically by `node.name()` so the cycle order is stable.
6. Read `_cycle_anchor_name`. If it does not match the current anchor's `node.name()`, reset cycle: set `_cycle_anchor_name = node.name()`, `_cycle_link_index = 0`, and call `_save_dag_position()`.
7. Clamp `_cycle_link_index` to be within bounds (in case a link was deleted since last call): `_cycle_link_index = _cycle_link_index % len(links)`.
8. Get `target_link = links[_cycle_link_index]`.
9. Zoom to the target link: `nukescripts.clear_selection_recursive()`, `target_link['selected'].setValue(True)`, `nuke.zoomToFitSelected()`.
10. Advance the index: `_cycle_link_index = (_cycle_link_index + 1) % len(links)`.
